# tourism_ai

## Vision

Create a i want to build an ai tourism platform. that delivers an exceptional user experience through modern technology and thoughtful design. tourism_ai aims to solve real user problems with a focus on reliability, performance, and usability.

## Tech Stack

- **Language:** Python
- **Framework:** LangChain / LlamaIndex
- **Llm_Providers:** OpenAI / Anthropic / Open Source
- **Vector_Store:** ChromaDB / Pinecone / Qdrant
- **Database:** PostgreSQL
- **Api:** FastAPI
- **Testing:** pytest
- **Deployment:** Docker / Modal / Railway
